Todo App에 오신 것을 환영합니다 📝

이 앱은 JWT 인증 기반의 간단한 Todo API를 제공합니다.

API 사용법 요약

  • POST /login → 로그인 (username, password로 access token 발급)
  • GET /todo → 본인의 Todo 목록 조회
  • POST /todo → 새로운 Todo 생성
  • PUT /todo/<id> → Todo 수정
  • DELETE /todo/<id> → Todo 삭제

Swagger 문서에서 API를 테스트해보세요!
